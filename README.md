# Enjoy-OMDB

### Requisites:

- Npm
- Postgres

### Installation:

1. Create a database: `createdb omdb` (the database needs to be called **omdb**)
2. `npm install`
3. Check running scripts in package.json
4. `npm start` to run front and back concurrently
